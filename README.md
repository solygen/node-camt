node-camt
=========

``parse csv-camt files and save data as json``

###Dependencies
[![dependencies](https://david-dm.org/solygen/node-camt.svg)](https://david-dm.org/solygen/node-camt)

###Usage
put your csv-camt files into

```
../data-source/
```

run

```
node ../lib/camt.js

```

use json stored in

```
../data-target/

```
